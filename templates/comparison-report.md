# Comparison Report Template

多工具横向对比报告模板。

## 使用方式

当用户要求对比 2+ 个工具时，使用此模板生成报告。

## 模板

```markdown
# ⚖️ <工具A> vs <工具B> vs <工具C> — <领域>工具对比

> 调研时间: YYYY-MM-DD | 方法: Safari 官网实测 | 由 AI Tool Scout 自动生成

## 一、一句话总结

| 工具 | 一句话 | 适合谁 |
|------|--------|--------|
| A | ... | ... |
| B | ... | ... |
| C | ... | ... |

> 🏆 综合推荐: <工具名>，理由：...

## 二、功能对比

| 功能 | A | B | C |
|------|:--:|:--:|:--:|
| 功能1 | ✅ | ❌ | ✅ |
| 功能2 | ✅ | ✅ | ❌ |
| ... | | | |

## 三、价格对比

| 工具 | 免费版 | 付费版 | 性价比 |
|------|--------|--------|:--:|
| A | 有/无 | $X/月 | ⭐⭐⭐ |
| B | 有/无 | $X/月 | ⭐⭐ |
| C | 有/无 | $X/月 | ⭐⭐⭐⭐ |

## 四、上手难度

| 工具 | 难度 | 学习曲线 | 文档质量 |
|------|:--:|:--:|:--:|
| A | 初级 | 10分钟 | 好 |
| B | 中级 | 30分钟 | 一般 |

## 五、选题建议

### 📊 内容策略

- 先做 A 的单独测评 → 表现好再做 VS 视频
- VS 视频标题: 「<A> vs <B>：谁才是最好的<领域>工具？」
```
