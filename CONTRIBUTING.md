# Contributing

欢迎贡献！

## 提 Issue

- 🐛 Bug：描述现象和复现步骤
- 💡 功能：描述场景和期望
- 📝 文档：哪里不清楚

## 提 PR

1. Fork → 建分支 → 改 → 提 PR
2. 确保 `python3 scripts/validate-skill.py` 通过（如果有的话）
3. 中文优先，用具体示例代替抽象描述

## 项目结构

```
ai-tool-scout/
├── SKILL.md           ← 核心 Skill
├── README.md          ← GitHub 首页
├── references/        ← 参考文档
├── templates/         ← 报告模板
├── examples/          ← 示例
└── assets/            ← 图片
```
